# File Explorer

Tree-based file browser for navigating project files within agent sessions.

## Features

- Expandable/collapsible directory tree
- File type icons (Swift, Python, JS, JSON, YAML, Markdown, etc.)
- Click to open files in the Editor
- Auto-refresh on file system changes

## File Icons

| Extension | Icon |
|-----------|------|
| `.swift` | `swift` |
| `.py` | `text.badge.star` |
| `.js`, `.ts` | `j.circle` / `t.circle` |
| `.json` | `curlybraces` |
| `.yml`, `.yaml` | `list.bullet` |
| `.md` | `doc.richtext` |
| Directory | `folder.fill` |
